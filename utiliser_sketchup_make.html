Comment utiliser Sketchup Make

Rédaction en cours !

Pourquoi Sketchup Make ?

Sketchup Make est un logiciel gratuit qui nous permet de concevoir des modèles 3D de poêles de masse.

En un mot, c'est MAGIQUE !

Coté conception, il nous permet de concevoir des poêles avec des vues 3D, des composants réutilisables, d'obtenir le débit des briques, etc..

Simple d'accès, il permet à ceux qui veulent simplement construire leur poêle d'avoir toutes les informations nécessaires pour le faire.

Le logiciel permet aussi de décomposer le poêle et de le voir étage par étage. L'outil "Mesure" permet de mesurer toutes les côtes nécessaires. La vue en 3D permet de comprendre très facilement le fonctionnement du poêle.

Télécharger et installer Sketchup Make

Il y a trois versions du logiciel Sketchup :

  1. Sketchup Make (gratuit, logiciel à installer)
  2. Sketchup Free (gratuit, logiciel en ligne)
  3. Sketchup Pro (payant)
Pour notre usage, Sketchup Make fait très bien l'affaire. Il n'y a pas besoin d'acheter une version Pro.

Si vous êtes sous Windows ou Mac :

Téléchargez sur cette page la dernière version de Sketchup Make (actuellement c'est la version 2017) et installez la.

Si vous êtes sur Linux :

Téléchargez Sketchup Make 2016 et installez le en suivant la procédure décrite ici (en anglais) : install-google-sketchup-make-2016-ubuntu-16-04.

Si vous n'y arrivez pas, vous pouvez aussi télécharger Sketchup 8 et l'installer avec Wine. Normalement la version 8 est bien supportée sous linux et elle devrait fonctionner correctement. L'installation de cette version est plus facile.

Une page d'explication sur l'installation de Sketchup 8 sous Ubuntu est disponible : https://doc.ubuntu-fr.org/sketchup.

Nous publions une version de nos plans sous Sketchup 8. Si vous ne la trouvez pas pour le plan que vous voulez, n'hésitez pas à nous contacter pour nous demander le plan en version Sketchup 8.

Si rien ne marche :

Essayez la version en ligne : Sketchup Free.

Utilisation du logiciel

Sketchup est de très loin le logiciel de conception 3D le plus simple et facile à prendre en main. Il est largement utilisé et si vous n'arrivez pas à vous en servir, demandez de l'aide à l'adolescent le plus proche ou allez voir des tutoriels sur Youtube.

Pour agrandir une image, positionnez le curseur sur l'image et faites clic droit > Afficher l'image.

NB : notre version est en anglais, les termes sont donc en anglais.

1. Choisir une "template" :

Lorsque vous ouvrez le logiciel pour la première fois, il vous demande de choisir une "template" : choisissez "Construction Documentation - Millimeters".

Sélectionnez "Construction Documentation - Millimeters" et cliquez sur "Start using Sketchup".

2. Afficher les "trays"


Pour lire un modèle 3D de poêle :

Vous n'avez besoin que d'un seul "tray" qui sera placé à droite de l'écran.

Pour ce faire, aller dans le menu "window", puis "default tray" et sélectionnez uniquement les éléments suivants :

  • Entity info
  • Materials
  • Layers

Le "default tray" avec les 3 menus importants est à droite.

Pour concevoir ou modifier un modèle 3D de poêle :

Vous avez besoin de deux "trays".

Pour ce faire, aller dans le menu "window", puis "New tray" et sélectionnez "Components".

Le "tray 1" avec les composants est à gauche.

3. Les calques

Les calques, c'est la fonction géniale de ce logiciel ! Un calque est un ensemble de composants (des briques d'un certain étage par exemple). On peut rendre un calque invisible en le décochant dans le "tray" à droite.

Tous les calques sont cochés : l'ensemble du poêle est visible.

Les calques de l'habillage, de la plancha et des portes sont décochés : on ne voit que le coeur de chauffe.

4. Obtenir le débit d'un poêle

Le débit d'un poêle correpond à la liste de tous les éléments nécessaires pour le construire.

Pour l'obtenir, aller dans le menu "window", puis "Model Info" et sélectionnez "Statistics" et "Only components". Là vous aurez une liste de tous les composants du modèle 3D : c'est le débit. Copiez cette liste et imprimez là.

La fenêtre pour obtenir le débit.

Faire le symmétrique d'un poêle

Vous voulez retourner un poêle ? Facile !

Sélectionnez tout le modèle en faisant Ctrl + a. Ensuite clic droit > Retourner selon > direction Rouge.

Le poêle est sélectionné.
